## Installation Commands

### Minimal Installation
```bash
pip install numpy pandas scipy matplotlib scikit-learn PyYAML pyoma2
```

### Full Installation  
```bash
pip install numpy pandas scipy matplotlib scikit-learn PyYAML pyoma2 seaborn plotly joblib
```

### Development Installation
```bash
pip install -r requirements-dev.txt
```

## Platform-Specific Notes

### Windows
- Ensure Visual Studio Build Tools are installed for compiling dependencies
- Use Anaconda/Miniconda for easier dependency management

### macOS  
- Xcode Command Line Tools may be required
- Consider using Homebrew for system dependencies

### Linux
- Development headers for BLAS/LAPACK may be needed
- Ubuntu/Debian: `sudo apt-get install python3-dev libblas-dev liblapack-dev`
- CentOS/RHEL: `sudo yum install python3-devel blas-devel lapack-devel`
